Primero, descarga todo el contenido en un archivo .zip y descomprímelo con la opcion "descomprimir aqui". 
Luego, haz clic derecho dentro de la carpeta y selecciona Abrir en terminal (o shell). Ejecuta el comando: py server.py 
Finalmente, mantén presionada la tecla Ctrl y haz clic en el enlace que aparecerá para abrir la aplicación en el puerto 8000.
